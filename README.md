# Dotfiles Repository

这是我的个人配置文件仓库，包含了一些基本的 shell 和编辑器配置。

## 包含的配置文件

- `.zshrc` - Zsh shell 配置
- `.vimrc` - Vim 编辑器配置
- `.tmux.conf` - Tmux 终端复用器配置

## 使用方法

运行初始化脚本：

```bash
./init.sh
```

该脚本会将仓库中的配置文件通过软链接的方式链接到你的家目录中。如果家目录中已存在同名文件，脚本会自动为其创建备份（添加 .bak 后缀）。

## 注意事项
这些配置文件仅满足我个人的使用习惯，可能不适合你的需求。使用前请先查看配置内容。
